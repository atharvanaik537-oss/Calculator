# Advanced Calculator Pro

## Overview:
**Advanced Calculator Pro*** is a premium, feature-rich web application that blends high-level mathematical functionality with a modern "Glassmorphism" aesthetic. Designed for both casual and scientific use, it provides a tactile, interactive experience with real-time history tracking, memory functions, and an adaptive layout that works perfectly on any device.

---
## Key Features:
**1. Modern Glassmorphism UI:** A sleek, translucent container set against a deep gradient background for a premium aesthetic.
**2. Color-Coded Controls:
* **Red:** Essential functions like Clear (AC), Delete (DEL), and Sign Toggle (+/-).
* **Orange:** Mathematical operators (+, -, ×, ÷).
* **Green:** The Equals (=) button for easy identification of the primary action.
* **Dark Blue:** Numeric inputs and decimal points.Scientific Red: Specialized functions like Square Root ($\sqrt{x}$) and Percentage (%).
**3. Dual-Line Display:** Features a primary result display and a smaller history line above it to track current expressions.
**4. Responsive and Tactile:** Buttons are designed with large touch/click targets and smooth hover transitions.

## Installation and Setup
To set up this project locally on your machine:

1. Clone the Repository:
```bash
git clone https://github.com/atharvanaik537-oss/Calculator.git
```
2. Navigate to the Folder:
```bash
cd Calculator
```
3. Check Files: Ensure that index.html, style.css, and script.js are all present in the root directory.

## Steps to Run the Website
You can run this project instantly using the same method shown in your screenshot:

**Method 1:** VS Code Live Server
Open the project folder in VS Code.
Right-click on index.html and select "Open with Live Server".
The calculator will launch at http://127.0.0.1:5500, as seen in your local environment.

**Method 2:** Direct Open
Locate your project folder in your computer's File Explorer.
Double-click on index.html to open it in your default web browser.
